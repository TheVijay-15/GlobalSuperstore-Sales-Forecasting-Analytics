# Global Superstore Sales Analytics
## Overview
### End-to-end data science project analyzing 51,290 retail transactions to deliver predictive sales forecasts, customer segmentation, and interactive dashboards.

## Technical Skills Demonstrated
### 1. Data Processing: Cleaned 51K transactions, handled missing values, engineered time-based features

### 2. Machine Learning: Built XGBoost forecasting model (R²=0.83) with 95% confidence intervals

### 3. Customer Analytics: RFM segmentation with K-Means clustering identifying 4 customer groups

### 4. Business Intelligence: Created 4-page Power BI dashboard with interactive visualizations

## Full Pipeline: Raw data → EDA → Feature Engineering → ML Models → Dashboard deployment

## Quick Results
### 1. 25% inventory cost reduction potential through improved forecasting

### 2. 4 customer segments identified (VIP, Loyal, At-Risk, New) for targeted marketing

### 3. 90-day sales forecasts with Prophet and XGBoost model comparison

### Business insights: December sales 2.3x higher, Technology category most profitable
